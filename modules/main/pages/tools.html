Miscellaneous Tools

The Semaphore project develops these tools to help researchers in handling and processing their data:

Kepler actors and examples

Kepler is a free software system for designing, executing, reusing, evolving, archiving, and sharing scientific workflows. The Semaphore project provides Kepler component (Actors) to run Century and Daycent models, and other components to help running the simulation and process the output data. Download the provided examples to learn on how to use Kepler to run the Century and Daycent models.

Microsoft Excel Addins

This addins for Microsoft Excel provides an interface to automate importing output data from Century/Daycent modelling to Excel spreadsheets for further processing and/or visualization. Instead of manually opening each output file and converting it to a sheet, use this tool to import all the available output files in a specified directory with a single click. Each file will be converted into a seperate sheet on the active workbook.
