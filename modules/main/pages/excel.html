Semaphore Addins for Microsoft Excel

This addins for Microsoft Excel provides an interface to automate importing output data from Century/Daycent modelling to Excel spreadsheets for further processing and/or visualization. Instead of manually opening each output file and converting it to a sheet, use this tool to import all the available output files in a specified directory with a single click. Each file will be converted into a seperate sheet on the active workbook.

Currently, the addins is only compatible with Windows version of Microsoft Excel.

Download semaphore.xlam (25kB)

Installation

  1. Put the semaphore.xlam file under C:\Users\[your_user_name]\AppData\Roaming\Microsoft\AddIns\ for Windows 7 or C:\Documents and Settings\[your_user_name]\Application Data\Microsoft for Windows XP
  2. Go to Excel Options, Add-Ins, Click Go... button. Tick the checkbox with Semaphore label, and press OK.

Usage

Go to Semaphore tab and click the Century/Daycent button.



Choose the desired model, only Century and Daycent are currently supported. Choose the directory that contains all the output files (*.lis, *.out, *.csv). Enter the name of the .lis output file (make sure the .lis extension is omitted), or select the file using the Browse button. Click the Import Data button to start the process. It will take some time to import all the output files to the spreadsheets.



Each output files will have its own sheet named exactly as the output file name. After a model re-run (ouput files has been changed), click the Refresh Data button to update the sheets with the current output files.
